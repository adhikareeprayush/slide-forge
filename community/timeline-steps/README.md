# Timeline Steps

A three-column timeline layout for showing process phases, milestones, or roadmaps.

## Slots

| Slot | Type | Description |
|------|------|-------------|
| `title` | heading | Slide title |
| `step1`–`step3` | subheading | Phase labels |
| `detail1`–`detail3` | body | Phase descriptions |
